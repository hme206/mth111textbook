Chapter 1: Numbers

Section 1.5: Estimation with Whole Numbers

Learning Objective(s)

  1. Use rounding to estimate sums and differences.
  2. Use rounding to estimate the solutions for application problems.

Introduction

An estimate is an answer to a problem that is close to the exact solution, but not necessarily exact. Estimating can come in handy in a variety of situations. For example, when buying a new computer setup — a tower and keyboard for \($\)1,295, a monitor for \($\)679, a printer for \($\)486, a warranty for \($\)196, and software for \($\)374 — estimating helps you know roughly how much you'll spend without adding all the numbers exactly.

Estimation usually requires rounding. When you round a number, you find a new number close to the original. Because rounded numbers have zeros in some place values, adding or subtracting them is easier, allowing you to find an estimate quickly. Estimating before calculating also lets you check whether your exact answer is reasonable.

Using Rounding to Estimate Sums and Differences

To estimate a sum, round each addend to the same place value and then add the rounded numbers together.

Example 1

Estimate the sum \(1{,}472 + 398 + 772 + 164\) by rounding each number to the nearest hundred.

Solution

First, round each number to the nearest hundred:

OriginalRounded to Nearest Hundred
1,472→1,500
398→400
772→800
164→200

Then add the rounded numbers:

\[ \begin{aligned} &1{,}500 \\ &\phantom{0,}400 \\ &\phantom{0,}800 \\ +\;&\phantom{0,}200 \\ \hline &2{,}900 \end{aligned} \]

Answer: The estimate is 2,900. (The exact sum is 2,806 — the estimate is close.)

Rounding to a lesser place value produces a more accurate estimate, but requires more steps. The example below shows the same sum rounded to the nearest ten.

Example 2

Estimate the sum \(1{,}472 + 398 + 772 + 164\) by rounding each number to the nearest ten.

Solution

Round each number to the nearest ten:

OriginalRounded to Nearest Ten
1,472→1,470
398→400
772→770
164→160

Then add the rounded numbers:

\[ \begin{aligned} &1{,}470 \\ &\phantom{0,}400 \\ &\phantom{0,}770 \\ +\;&\phantom{0,}160 \\ \hline &2{,}800 \end{aligned} \]

Answer: The estimate is 2,800. (The exact sum is 2,806 — only 6 away.)

Self Check A

In three months, a freelance graphic artist earns \($\)1,290 for illustrating comic books, \($\)2,612 for designing logos, and \($\)4,175 for designing websites. Estimate how much she earned in total by first rounding each number to the nearest hundred.

Round each amount: \(\$1{,}290 \rightarrow \$1{,}300\),   \(\$2{,}612 \rightarrow \$2{,}600\),   \(\$4{,}175 \rightarrow \$4{,}200\).

\[\$1{,}300 + \$2{,}600 + \$4{,}200 = \$8{,}100\]

Estimated total: $8,100

You can also estimate differences using the same rounding approach.

Example 3

Estimate the difference of 5,876 and 4,792 by first rounding each number to the nearest hundred.

Solution

OriginalRounded to Nearest Hundred
5,876→5,900
4,792→4,800
\[ \begin{aligned} &5{,}900 \\ -\;&4{,}800 \\ \hline &1{,}100 \end{aligned} \]

Answer: The estimate is 1,100. (The exact difference is 1,084 — the estimate is 16 greater.)

Self Check B

Estimate the difference of 474,128 and 262,767 by rounding to the nearest thousand.

Round each number: \(474{,}128 \rightarrow 474{,}000\) and \(262{,}767 \rightarrow 263{,}000\).

\[474{,}000 - 263{,}000 = 211{,}000\]

Estimated difference: 211,000

Solving Application Problems by Estimating

Estimating is especially handy when you want to make sure you have enough money for several purchases, or when you need a quick check on a total.

Example 4

When buying a new computer, you find that the computer tower and keyboard cost \($\)1,295, the monitor costs \($\)679, the printer costs \($\)486, the 2-year warranty costs \($\)196, and a software package costs \($\)374. Estimate the total cost by first rounding each number to the nearest hundred.

Solution

Item CostRounded to Nearest Hundred
$1,295→$1,300
$679→$700
$486→$500
$196→$200
$374→$400
\[ \begin{aligned} &1{,}300 \\ &\phantom{0,}700 \\ &\phantom{0,}500 \\ &\phantom{0,}200 \\ +\;&\phantom{0,}400 \\ \hline &3{,}100 \end{aligned} \]

Answer: The total cost is approximately $3,100.

Example 5

James travels 3,247 m to the park, then 582 m to the store, then 1,634 m back to his house. Find the total distance traveled by first rounding each number to the nearest ten.

Solution

DistanceRounded to Nearest Ten
3,247 m→3,250 m
582 m→580 m
1,634 m→1,630 m
\[ \begin{aligned} &3{,}250 \\ &\phantom{0,}580 \\ +\;&1{,}630 \\ \hline &5{,}460 \end{aligned} \]

Answer: The total distance traveled was approximately 5,460 meters. (The exact sum is 5,463 m — only 3 away.)

Example 6

One mountain is 10,496 feet high and another is 7,421 feet high. Find the difference in height by first rounding each number to the nearest hundred.

Solution

HeightRounded to Nearest Hundred
10,496 ft→10,500 ft
7,421 ft→7,400 ft
\[ \begin{aligned} &10{,}500 \\ -\;&\phantom{0}7{,}400 \\ \hline &\phantom{0}3{,}100 \end{aligned} \]

Answer: The estimated difference in height is 3,100 feet. (The exact difference is 3,075 ft — the estimate is 25 greater.)

Self Check C

A space shuttle traveling at 17,581 miles per hour decreases its speed by 7,412 miles per hour. Estimate the speed of the space shuttle after it has slowed down by rounding each number to the nearest hundred.

Round each speed: \(17{,}581 \rightarrow 17{,}600\) and \(7{,}412 \rightarrow 7{,}400\).

\[17{,}600 - 7{,}400 = 10{,}200 \text{ mi/h}\]

Estimated speed: 10,200 mi/h

Summary

Estimation is very useful when an exact answer is not required. You can use estimation for problems related to travel, finances, and data analysis. Estimating is done by rounding each number to the same place value before adding or subtracting — this makes the arithmetic easier and produces an answer close to the exact value. Estimating before calculating also helps you catch errors in your work.
